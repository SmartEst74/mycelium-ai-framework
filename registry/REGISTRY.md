# Agency-Agent Registry

Imported from [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents).
179 specialized agent roles with personalities, processes, and proven deliverables.

## Usage

When Army Ant needs to build a team:
1. Look up the required capability in this registry
2. Load the agent role definition from `registry/roles/<department>/<role>.md`
3. Assign the best model for the task type (see MODEL_MATRIX)
4. Spawn a Dynamic Ant with the role persona + model + task
5. Dynamic Ant completes task, reports back, dies
